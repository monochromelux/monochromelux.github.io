glMatrixMode

선언 형태

  • void glMatrixMode ( GLenum mode );

매개 변수

변수명 설명
mode 연산을 수행할 행렬 스택의 종류를 지정합니다. 행렬 스택의 종류에는 GL_MODELVIEW, GL_PROJECTION, GL_TEXTURE가 있습니다.
기본값은 GL_MODELVIEW입니다. 또한 'ARB_imaging' 확장 기능이 지원되는 경우 GL_COLOR도 허용됩니다.
