glutTimerFunc

선언 형태

  • void glutTimerFunc ( unsigned int msecs, void (*func)( int value ), int value )

매개 변수

변수명 설명
msecs callback 함수를 호출할 간격을 밀리초 단위로 설정합니다.
func 호출할 Timer callback을 설정합니다.
value Timer callback으로 전달할 정숫값을 설정합니다.
