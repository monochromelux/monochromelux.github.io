glutInit

선언 형태

  • void glutInit ( int *argcp, char **argv );

매개 변수

변수명 설명
argcp 메인 함수의 원본 argc 변수에 대한 포인터입니다.
glutInit은 GLUT 라이브러리를 위한 명령줄 옵션을 추츨하므로, 값이 반환될 때 argcp에 의래 가리켜지는 값이 업데이트될 것입니다.
argv 프로그램 메인 함수의 argv 변수 원본입니다.
argcp의 경우처럼, glutInit은 GLUT 라이브러리에 의해 해석되는 모든 명령줄 옵션을 추출하므로 argv의 데이터가 업데이트될 것입니다.
