glBegin & glEnd

선언 형태

  • void glBegin ( GLenum mode );
  • void glEnd ( void );

매개 변수

변수명 설명
mode glBegin과 바로 다음 glEnd 사이에 있는 좌표들의 값을 읽어서 새로운 요소를 생성합니다.
요소의 형태는 지정된 다음 10가지 기호 상수를 통해 표현할 수 있습니다.: GL_POINTS, GL_LINES, GL_LINE_STRIP, GL_LINE_LOOP, GL_TRIANGLES, GL_TRIANGLE_STRIP, GL_TRIANGLE_FAN, GL_QUADS, GL_QUAD_STRIP, GL_POLYGON
