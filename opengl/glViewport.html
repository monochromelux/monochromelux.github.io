glViewport

선언 형태

  • void glViewport ( GLint x, GLint y, GLsizei width, GLsizei height );

매개 변수

변수명 설명
x, y 왼쪽 아래를 기준으로 뷰포트의 시작 위치를 픽셀 단위로 지정합니다.
기본값은 (0, 0)입니다.
width, height 뷰포트의 가로, 세로 크기를 지정합니다.
실행 창이 처음 실행되었을 때 width와 height는 해당 실행 창을 따라 설정됩니다.
