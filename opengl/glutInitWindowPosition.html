glutInitWindowPosition & glutInitWindowSize

선언 형태

  • void glutInitWindowPosition ( int width, int height );
  • void glutInitWindowSize ( int x, int y );

매개 변수

변수명 설명
width 창의 가로 길이를 픽셀 단위로 설정합니다.
height 창의 세로 길이를 픽셀 단위로 설정합니다.
x 창의 x 위치를 픽셀 단위로 설정합니다.
y 창의 y 위치를 픽셀 단위로 설정합니다.
