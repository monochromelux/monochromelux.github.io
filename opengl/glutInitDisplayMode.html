glutInitDisplayMode

선언 형태

  • void glutInitDisplayMode ( unsigned int mode );

매개 변수

변수명 설명
mode 설정할 GLUT 디스플레이 모드를 OR 비트 연산으로 입력합니다.
  • GLUT_RGBA: RGBA 모드를 사용합니다. GLUT_INDEX가 지정되지 않았을 경우 기본값에 해당합니다.
  • GLUT_RGB: GLUT_RGBA의 다른 이름입니다.
  • GLUT_SINGLE: 싱글 버퍼 모드를 사용합니다. GLUT_DOUBLE이 지정되지 않았을 경우 기본값에 해당합니다.
  • GLUT_DOUBLE: 더블 버퍼 모드를 사용합니다.
  • GLUT_DEPTH: 깊이 버퍼를 사용합니다.
