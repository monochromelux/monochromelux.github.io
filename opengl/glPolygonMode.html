glPolygonMode

선언 형태

  • void glPolygonMode ( GLenum face, GLenum mode );

매개 변수

변수명 설명
face mode가 적용될 폴리곤의 종류를 지정합니다.
앞면에 적용하기 위해서는 GL_FRONT, 뒷면에 적용하기 위해서는 GL_BACK, 앞면과 뒷면 모두에 적용하기 위해서는 GL_FRONT_AND_BACK을 사용합니다.
mode 폴리곤을 래스터화할 방법을 지정합니다.
GL_POINT, GL_LINE, GL_FILL을 사용할 수 있습니다. 기본값은 GL_FILL입니다.
